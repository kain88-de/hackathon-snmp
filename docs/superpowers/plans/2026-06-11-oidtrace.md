# OIDTrace Implementation Plan (contract level)

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax.

**Goal:** Implement the OIDTrace CLI (`walk`) per `docs/superpowers/specs/2026-06-11-oidtrace-architecture-design.md`, emitting traces that conform to `docs/trace-format.md` / `docs/trace-format.schema.json`.

**How to read this plan:** it specifies _contracts_ (module boundaries, data structures, signatures, behavioral test intentions) ŌĆö not implementations. You design the code, TDD-style, under strict tooling gates. Interfaces and assertions named here are binding; everything else is yours. `experiments/poc_roundtrip.py` is a **proven reference** for the BER codec, emulator, and walk mechanics ŌĆö consult it, do not copy it blindly (it predates several format decisions: no raw fields, no scrubber, GetBulk-only).

**Per-task discipline (applies to every task):** write the failing tests first ŌåÆ run them, confirm they fail for the right reason ŌåÆ implement to the contract ŌåÆ `just ci` green (format, lint, types, tests) ŌåÆ commit (separate `git add` then `git commit`).

**Out of scope (per spec):** SNMP v1, system-info allowlist, `show`, interactive community prompt, matrix CLI, anything touching packet bytes.

## Architecture (binding)

```
oidtrace/src/oidtrace/
  oid.py        @dataclass(frozen=True, order=True, slots=True) class Oid with a
                single field arcs: tuple[int, ...]; Oid.from_str("1.3.6.1") -> Oid
                (ValueError on garbage; ASCII digits, no leading zeros);
                str(oid) -> "1.3.6.1"; readable __repr__; oid.in_subtree(root) -> bool.
                order=True over arcs gives correct OID ordering (and bisect
                compatibility); frozen gives hashing.
  ber.py        minimal BER: tlv(tag, payload), encode_int(v, tag=0x02),
                encode_oid(Oid), read_tlv(buf, i)->(tag, payload: bytes, next_i)
                raising ValueError on any truncation/overrun, decode_int, decode_oid
                (payload-slices, not indices: every consumer slices immediately and
                tests assert on bytes directly)
  codec.py      encode_getbulk(request_id, oid, non_repeaters, max_repetitions,
                community=b"public")->bytes;
                encode_response(request_id, varbinds: Sequence[(Oid, tag, value_bytes)],
                ...)->bytes (used by the test emulator);
                decode_message(raw)->Message | Malformed  (tolerant: never raises);
                Varbind(oid, tag, value) frozen, with vtype/vlen properties;
                Message(pdu_tag, request_id, f1, f2, varbinds, raw);
                Malformed(raw, error); TAG_NAMES per format spec ┬¦ 5,
                unknown tags -> "tag:0xNN"; EXCEPTION_TAGS = {0x80, 0x81, 0x82}.
                f1/f2 stay generic: the one decoder serves walker responses
                (error_status/error_index) and emulator-received requests
                (non_repeaters/max_repetitions)
  violations.py check_exchange(*, sent_id, returned_id, prev_oid, varbinds,
                response_raw, strays)->list[str] ŌĆö pure; vocabulary:
                request-id-mismatch, oid-not-increasing, duplicate-response
                (malformed-ber is added by the walker)
  records.py    builder functions returning plain dicts (the schema is the contract;
                no dataclass mirror): header_record, exchange_record, event_record,
                summary_record, system_info_record (builder exists for format
                completeness; v1 walker never calls it). Varbind values must be
                unreachable from any built record.
  tracefile.py  TraceWriter(path): write(dict) appends one JSON line to gzip and
                flushes; context manager. read_trace(path)->Iterator[dict]: yields
                complete records, stops quietly at truncation (EOFError/BadGzipFile/
                partial last line).
  transport.py  Attempt(sent_at, received_at, error=None);
                ExchangeIO(attempts, response: (received_at, raw)|None, strays:
                list[(received_at, raw)]);
                Transport = typing.Protocol with
                async exchange(raw, *, timeout_s, retries)->ExchangeIO ŌĆö the walker
                depends on this protocol, never on the concrete class (ExchangeIO is
                plain data, so a FakeTransport replaying scripted ExchangeIO values is
                the walker's unit-test fixture);
                UdpTransport.create(host, port, rel: Callable[[], float]) (classmethod,
                asyncio datagram endpoint, connected socket); close();
                exchange(raw, *, timeout_s, retries)->ExchangeIO ŌĆö sends, waits,
                retries with byte-identical resend; never validates; ICMP errors via
                error_received -> attempt.error ("icmp-port-unreachable",
                "icmp-host-unreachable", "send-failed"); after a response, drain
                immediately-available datagrams into strays.
  walker.py     WalkSettings(bulk_size>=1 default 10, timeout_s=2.0, retries=2,
                start_oid="1.3.6.1", time_budget_s=None, give_up_after=3,
                community=b"public");
                run_walk(host, port, *, settings, path, label=None, session_id=None,
                run=1, runs_total=1, on_record=None)->str(end_reason) ŌĆö owns
                t0=time.monotonic(), all times round(rel, 6); every record goes
                through one emit() (file write + on_record callback);
                GetBulk-only loop; termination per spec: endOfMibView -> completed,
                left subtree -> completed, oid-not-increasing -> oid-loop event +
                end, time budget -> event + end, give_up_after consecutive
                response-less exchanges -> unresponsive; CancelledError -> summary
                "interrupted", re-raise.
  cli.py        main(argv)->int; subcommand walk(host, --port, --out, --label,
                --bulk-size, --timeout, --retries, --start-oid, --time-budget);
                DNS resolved up front (exit 2, message on stderr, no trace file);
                community from OIDTRACE_COMMUNITY env (default "public");
                live progress on stderr via on_record; terminal summary + trace path
                on stdout; exit 0 incl. KeyboardInterrupt.

oidtrace/tests/
  conftest.py            session fixture `record_validator` loading
                         ../docs/trace-format.schema.json (Draft 2020-12)
  unit/                  no I/O: test_oid, test_ber, test_codec_encode,
                         test_codec_decode, test_violations, test_records,
                         test_tracefile, test_walker_logic (FakeTransport)
  integration/           loopback UDP: conftest (async `emulator_factory` fixture),
                         test_emulator_smoke, test_transport, test_walker, test_cli,
                         test_reference_tools (marker `reference_tools`)
  support/emulator.py    scripted quirk emulator (asyncio DatagramProtocol, reuses
                         codec): EmuDevice(tree: [(Oid, tag, vlen)], quirks),
                         EmuDevice.simple(n_oids, quirks=None) builds an ifTable-like
                         tree; Quirks(fixed_request_id, end_of_mib: "eom"|"silence"|
                         "wrap", duplicate_responses, slow_prefix+per_oid_delay_s,
                         drop_all)
```

**Binding data-structure decisions:** OIDs are a frozen dataclass over
`arcs: tuple[int, ...]` (string comparison mis-orders `1.3.6.1.10` vs `1.3.6.1.2`;
tuple-based ordering compares correctly ŌĆö test this). Strings only at boundaries via
`Oid.from_str` / `str(oid)`. Tolerant decode is a union return, not exceptions. Records are dicts from
builders, schema-validated in tests. Varbind value bytes exist in memory only.

**Everything crossing a boundary is a frozen dataclass with immutable fields:**
`Oid`, `Varbind`, `Message` (`varbinds: tuple[Varbind, ...]`), `Malformed`, `Attempt`
(constructed once with its final outcome ŌĆö the transport never mutates an attempt
mid-retry), `ExchangeIO` (tuple fields ŌĆö FakeTransport fixtures are safely shareable
across tests), `WalkSettings`, `Quirks`, `EmuDevice`. Records stay plain dicts
(serialized the moment they are built; the schema is their contract). Mutable state
exists in exactly two places: the walker's loop state and the transport's internal
event queue.

**Runtime data flow (one I/O point, everything else pure):** per exchange, the walker
ŌĆö the only orchestrator, strictly sequential, sole owner of state (cursor, seq, t0,
violation counts) ŌĆö runs:

```
codec.encode_getbulk(...)          pure: values -> bytes
await transport.exchange(...)      THE ONLY I/O: bytes -> ExchangeIO
codec.decode_message(...)          pure: bytes -> Message | Malformed
violations.check_exchange(...)     pure: values -> [str]
records.exchange_record(...)       pure: values -> dict
emit(dict) -> TraceWriter + on_record   dumb sinks; nothing flows back
termination decision + cursor advance   pure, from decoded varbinds
```

Knowledge boundaries: transport treats packets as opaque bytes (no SNMP ŌĆö that is why
it cannot accidentally validate); codec knows SNMP but never sees a socket; violations,
records are valueŌåÆvalue; tracefile knows JSON+gzip, not SNMP. Import DAG:
`oid ŌåÉ ber ŌåÉ codec ŌåÉ {violations, walker, emulator}`; records/tracefile/transport are
independent legs; walker is the join; cli sits on top. The emulator mirrors the same
layering server-side (decode ŌåÆ behavior source ŌåÆ encode), sharing only the codec.

Errors become data at each boundary going up: ber raises ValueError ŌåÆ codec catches
into `Malformed` ŌåÆ transport maps OS errors to `Attempt.error` strings ŌåÆ walker turns
everything into records plus an `end_reason` ŌåÆ cli maps that to exit codes. Device
misbehavior never propagates as an exception ("misbehavior is data", structurally).

**Where we test:** the trace file is the primary observable ŌĆö walker tests assert on
written records, never internals. Wire bytes are the second contract ŌĆö codec output
must decode via pysnmp **spec-driven** (`decoder.decode(raw, asn1Spec=Message())`;
pyasn1's schemaless decoder cannot walk SNMP PDUs ŌĆö PoC lesson).

## Task 1: Toolchain-first scaffolding (strict from commit zero)

## Task 11: `walker.py`

- [ ] **Unit tests first** (`tests/unit/test_walker_logic.py`, no sockets): a

      endOfMibView ŌåÆ completed, left-subtree ŌåÆ completed, non-increasing ŌåÆ oid-loop,
      give_up_after counting across response-less exchanges (incl. recovery resetting
      the counter), malformed response ŌåÆ malformed-ber violation + malformed record,
      ICMP attempt errors recorded. Assert on the written trace records.
- [ ] **Integration tests** (real UdpTransport + emulator) assert **only on the written
      trace** (and the returned end_reason); every record of every test trace must
      validate against the schema:
  - clean 50-OID walk ŌåÆ `completed`; header first / summary last; summary
    `oids_seen == 50`; distinct OIDs in responses == 50.
  - `fixed_request_id=1` device ŌåÆ walk **completes** (the point of the tool);
    every exchange carries `violations: ["request-id-mismatch"]` and
    `response.request_id == 1`; summary violation_counts matches.
  - `end_of_mib="wrap"` ŌåÆ `oid-loop`: an `event` record `oid-loop-detected` exists,
    summary end_reason matches.
  - `drop_all` + `give_up_after=2` ŌåÆ `unresponsive`, exactly 2 exchange records, no
    `response` keys, each with retries+1 attempts.
  - slow device + `time_budget_s` ŌåÆ `time-budget-exceeded` event + end_reason.
  - cancellation mid-walk ŌåÆ file ends with summary `end_reason: "interrupted"`.
  - `on_record` callback receives exactly the records written, in order
    (`streamed == list(read_trace(path))`).
