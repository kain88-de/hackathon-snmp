# OIDTrace Implementation Plan (contract level)

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax.

**Goal:** Implement the OIDTrace CLI (`walk`) per `docs/superpowers/specs/2026-06-11-oidtrace-architecture-design.md`, emitting traces that conform to `docs/trace-format.md` / `docs/trace-format.schema.json`.

**How to read this plan:** it specifies _contracts_ (module boundaries, data structures, signatures, behavioral test intentions) — not implementations. You design the code, TDD-style, under strict tooling gates. Interfaces and assertions named here are binding; everything else is yours. `experiments/poc_roundtrip.py` is a **proven reference** for the BER codec, emulator, and walk mechanics — consult it, do not copy it blindly (it predates several format decisions: no raw fields, no scrubber, GetBulk-only).

**Per-task discipline (applies to every task):** write the failing tests first → run them, confirm they fail for the right reason → implement to the contract → `just ci` green (format, lint, types, tests) → commit (separate `git add` then `git commit`).

**Out of scope (per spec):** SNMP v1, system-info allowlist, `show`, interactive community prompt, matrix CLI, anything touching packet bytes.

## Architecture (binding)

```
oidtrace/src/oidtrace/
  oid.py        Oid = tuple[int, ...]; parse_oid(str)->Oid (ValueError on garbage),
                format_oid(Oid)->str, in_subtree(root, oid)->bool
  ber.py        minimal BER: tlv(tag, payload), encode_int(v, tag=0x02),
                encode_oid(Oid), read_tlv(buf, i)->(tag, vstart, vend) raising
                ValueError on any truncation/overrun, decode_int, decode_oid
  codec.py      encode_getbulk(request_id, oid, non_repeaters, max_repetitions,
                community=b"public")->bytes;
                encode_response(request_id, varbinds: Sequence[(Oid, tag, value_bytes)],
                ...)->bytes (used by the test emulator);
                decode_message(raw)->Message | Malformed  (tolerant: never raises);
                Varbind(oid, tag, value) frozen, with vtype/vlen properties;
                Message(pdu_tag, request_id, f1, f2, varbinds, raw);
                Malformed(raw, error); TAG_NAMES per format spec § 5,
                unknown tags -> "tag:0xNN"; EXCEPTION_TAGS = {0x80, 0x81, 0x82}
  violations.py check_exchange(*, sent_id, returned_id, prev_oid, varbinds,
                response_raw, strays)->list[str] — pure; vocabulary:
                request-id-mismatch, oid-not-increasing, duplicate-response
                (malformed-ber is added by the walker)
  records.py    builder functions returning plain dicts (the schema is the contract;
                no dataclass mirror): header_record, exchange_record, event_record,
                summary_record, system_info_record (builder exists for format
                completeness; v1 walker never calls it). Varbind values must be
                unreachable from any built record.
  tracefile.py  TraceWriter(path): write(dict) appends one JSON line to gzip and
                flushes; context manager. read_trace(path)->Iterator[dict]: yields
                complete records, stops quietly at truncation (EOFError/BadGzipFile/
                partial last line).
  transport.py  Attempt(sent_at, received_at, error=None);
                ExchangeIO(attempts, response: (received_at, raw)|None, strays:
                list[(received_at, raw)]);
                UdpTransport.create(host, port, rel: Callable[[], float]) (classmethod,
                asyncio datagram endpoint, connected socket); close();
                exchange(raw, *, timeout_s, retries)->ExchangeIO — sends, waits,
                retries with byte-identical resend; never validates; ICMP errors via
                error_received -> attempt.error ("icmp-port-unreachable",
                "icmp-host-unreachable", "send-failed"); after a response, drain
                immediately-available datagrams into strays.
  walker.py     WalkSettings(bulk_size>=1 default 10, timeout_s=2.0, retries=2,
                start_oid="1.3.6.1", time_budget_s=None, give_up_after=3,
                community=b"public");
                run_walk(host, port, *, settings, path, label=None, session_id=None,
                run=1, runs_total=1, on_record=None)->str(end_reason) — owns
                t0=time.monotonic(), all times round(rel, 6); every record goes
                through one emit() (file write + on_record callback);
                GetBulk-only loop; termination per spec: endOfMibView -> completed,
                left subtree -> completed, oid-not-increasing -> oid-loop event +
                end, time budget -> event + end, give_up_after consecutive
                response-less exchanges -> unresponsive; CancelledError -> summary
                "interrupted", re-raise.
  cli.py        main(argv)->int; subcommand walk(host, --port, --out, --label,
                --bulk-size, --timeout, --retries, --start-oid, --time-budget);
                DNS resolved up front (exit 2, message on stderr, no trace file);
                community from OIDTRACE_COMMUNITY env (default "public");
                live progress on stderr via on_record; terminal summary + trace path
                on stdout; exit 0 incl. KeyboardInterrupt.

oidtrace/tests/
  conftest.py            session fixture `record_validator` loading
                         ../docs/trace-format.schema.json (Draft 2020-12)
  unit/                  no I/O: test_oid, test_ber, test_codec_encode,
                         test_codec_decode, test_violations, test_records,
                         test_tracefile
  integration/           loopback UDP: conftest (async `emulator_factory` fixture),
                         test_emulator_smoke, test_transport, test_walker, test_cli,
                         test_reference_tools (marker `reference_tools`)
  support/emulator.py    scripted quirk emulator (asyncio DatagramProtocol, reuses
                         codec): EmuDevice(tree: [(Oid, tag, vlen)], quirks),
                         EmuDevice.simple(n_oids, quirks=None) builds an ifTable-like
                         tree; Quirks(fixed_request_id, end_of_mib: "eom"|"silence"|
                         "wrap", duplicate_responses, slow_prefix+per_oid_delay_s,
                         drop_all)
```

**Binding data-structure decisions:** OIDs are `tuple[int, ...]` internally (string
comparison mis-orders `1.3.6.1.10` vs `1.3.6.1.2`; tuples compare correctly — test
this). Tolerant decode is a union return, not exceptions. Records are dicts from
builders, schema-validated in tests. Varbind value bytes exist in memory only.

**Where we test:** the trace file is the primary observable — walker tests assert on
written records, never internals. Wire bytes are the second contract — codec output
must decode via pysnmp **spec-driven** (`decoder.decode(raw, asn1Spec=Message())`;
pyasn1's schemaless decoder cannot walk SNMP PDUs — PoC lesson).

## Task 1: Toolchain-first scaffolding (strict from commit zero)

- [ ] Root `pyproject.toml`: uv workspace, member `oidtrace`.
- [ ] `oidtrace/pyproject.toml`: project (requires-python >=3.13, no runtime deps,
      script `oidtrace = "oidtrace.cli:main"`, hatchling); dev group: pytest,
      pytest-asyncio, jsonschema, pysnmp, ruff, pyrefly, pyright.
- [ ] **Strict tooling config** (this is the point of the task):
  - ruff: `line-length = 100`, lint.select =
    `["E","F","W","I","N","UP","B","A","C4","SIM","ARG","RUF","ASYNC","TC","PTH","PL"]`
    with only narrowly-justified per-rule ignores (e.g. `PLR2004` in tests); fix
    findings, don't blanket-ignore.
  - **pyrefly** is the CI type checker: configure `[tool.pyrefly]` in
    `oidtrace/pyproject.toml` so it finds `src/` and `tests/` and resolves the
    **workspace venv** at the repo root (`uv sync` creates `../.venv`). Acceptance:
    `uv run pyrefly check` passes on the skeleton and fails on a deliberately
    mistyped scratch function (verify both, then delete the scratch).
  - **pyright** for the editor/agent: `[tool.pyright] venvPath = ".." venv = ".venv"`,
    `typeCheckingMode = "strict"`. Acceptance: `uv run pyright src` clean on the
    skeleton — no import-resolution false positives.
- [ ] `oidtrace/Justfile`: `fmt` (ruff format), `lint` (ruff check), `types` (pyrefly
      check, plus a `types-pyright` recipe), `test` (pytest -m "not reference_tools"
      -q), `test-all` (REQUIRE_REFERENCE_TOOLS=1 pytest -q), `ci: fmt lint types test`.
- [ ] pytest: `asyncio_mode = "auto"`, marker `reference_tools`, testpaths `tests`.
- [ ] Package skeleton (`src/oidtrace/__init__.py`, test package dirs, `conftest.py`
      with the `record_validator` fixture), one smoke test importing the package.
- [ ] `uv sync && just ci` green → commit.

## Task 2: `oid.py`

- [ ] Tests: parse/format round-trip; **tuple ordering beats string ordering**
      (`parse_oid("1.3.6.1.2") < parse_oid("1.3.6.1.10")`); `in_subtree` (inside,
      outside, equal); parse rejects `""`, `"1.3.x"`, `"1..3"`, `".1.3"` with
      ValueError.
- [ ] Implement → `just ci` → commit.

## Task 3: `ber.py`

- [ ] Tests: TLV short + long-form lengths (payload 10 and 300 bytes; check the exact
      length bytes); int round-trip over {0, 1, 127, 128, 255, 1042, 2^31−1}; OID
      round-trip including multi-byte arcs (`...2636...`, instance `123456`);
      `read_tlv` raises ValueError on truncated header, truncated body, and bad
      long-form length.
- [ ] Implement → `just ci` → commit.

## Task 4: `codec.py` encode side

- [ ] Tests: `encode_getbulk` (bulk 10 and bulk 1) and `encode_response` outputs decode
      via pysnmp spec-driven with empty remainder (handle both pysnmp API spellings:
      `PROTOCOL_MODULES`/`protoModules`, `SNMP_VERSION_2C`/`protoVersion2c`);
      max-repetitions value is visible on the wire; community is settable.
- [ ] Implement → `just ci` → commit.

## Task 5: `codec.py` tolerant decode

- [ ] Tests: well-formed response round-trip (request_id, varbind oid/vtype/vlen);
      **returned request-id preserved as-is** (encode with rid 1, decode shows 1 —
      caller compares); unknown value tag → `"tag:0x99"`, not an error; garbage inputs
      (`b""`, `b"\xff"`, overlong length claim, ASCII text) → `Malformed` with the raw
      and a non-empty error, never an exception.
- [ ] Implement → `just ci` → commit.

## Task 6: `violations.py`

- [ ] Tests: request-id mismatch detected; non-increasing OID detected (equal counts as
      non-increasing; exception tags 0x80–0x82 skipped); duplicate detected only when a
      stray equals the response bytes; clean exchange → `[]`.
- [ ] Implement (pure function) → `just ci` → commit.

## Task 7: `records.py`

- [ ] Tests: every builder's output validates against the schema (use realistic field
      values incl. a stray, a violation, an attempt with `error`); **a Varbind carrying
      `b"SECRET-VALUE"` never appears in any serialized record** (json.dumps scan), and
      its varbind dict is exactly `{oid, vtype, vlen}`; an exchange with
      `response_fields=None` has no `response` key and still validates; `malformed`
      records carry `{error, length}`.
- [ ] Implement → `just ci` → commit.

## Task 8: `tracefile.py`

- [ ] Tests: write→read round-trip; **truncation tested, not assumed** — chop bytes off
      the end of a written gzip, earlier complete records still read; a record is
      readable from the file immediately after `write()` without `close()` (Ctrl-C
      safety).
- [ ] Implement → `just ci` → commit.

## Task 9: quirk emulator (test support)

- [ ] `tests/support/emulator.py` + `emulator_factory` async-contextmanager fixture in
      `tests/integration/conftest.py` (binds 127.0.0.1:0, yields (host, port), closes
      on exit).
- [ ] Smoke tests (raw asyncio datagram client, no transport.py yet): a GetBulk gets a
      response with the requested count of varbinds and echoed request-id;
      `fixed_request_id=1` quirk overrides the echoed id.
- [ ] `just ci` → commit.

## Task 10: `transport.py`

- [ ] Tests (against the fixture): response recorded with `received_at > sent_at >= 0`
      and exactly one attempt; `drop_all` device → `response is None`, retries+1
      attempts all unanswered; `duplicate_responses` → exactly one stray whose bytes
      equal the response (transport keeps bytes in memory; the trace will not);
      sending to a **closed local port** → `attempt.error == "icmp-port-unreachable"`
      and no response.
- [ ] Implement (asyncio protocol with an event queue; timeout via `asyncio.timeout`)
      → `just ci` → commit.

## Task 11: `walker.py`

- [ ] Tests assert **only on the written trace** (and the returned end_reason); every
      record of every test trace must validate against the schema:
  - clean 50-OID walk → `completed`; header first / summary last; summary
    `oids_seen == 50`; distinct OIDs in responses == 50.
  - `fixed_request_id=1` device → walk **completes** (the point of the tool);
    every exchange carries `violations: ["request-id-mismatch"]` and
    `response.request_id == 1`; summary violation_counts matches.
  - `end_of_mib="wrap"` → `oid-loop`: an `event` record `oid-loop-detected` exists,
    summary end_reason matches.
  - `drop_all` + `give_up_after=2` → `unresponsive`, exactly 2 exchange records, no
    `response` keys, each with retries+1 attempts.
  - slow device + `time_budget_s` → `time-budget-exceeded` event + end_reason.
  - cancellation mid-walk → file ends with summary `end_reason: "interrupted"`.
  - `on_record` callback receives exactly the records written, in order
    (`streamed == list(read_trace(path))`).
- [ ] Implement → `just ci` → commit.

## Task 12: `cli.py`

- [ ] Tests (sync; run the emulator on a background thread with its own loop since
      `main()` calls `asyncio.run`): successful walk → exit 0, one
      `*.oidtrace.jsonl.gz` in `--out`, header carries the label, stdout contains the
      end_reason and "exchanges"; unresolvable host → exit 2, stderr mentions
      resolution, **no trace file written**; community comes from `OIDTRACE_COMMUNITY`
      (autouse fixture sets it).
- [ ] Implement → `just ci` → commit.

## Task 13: net-snmp cross-walk (reference tool)

- [ ] Test (marker `reference_tools`): skip with a visible message when `snmpbulkwalk`
      is absent, **fail** instead when `REQUIRE_REFERENCE_TOOLS=1`; run
      `snmpbulkwalk -v2c -c public -On -Cr10` against the emulator (subprocess in an
      executor) and our walker against the same emulator; the reference OID sequence
      must be a prefix of ours and our distinct-OID count must equal the device tree.
- [ ] Verify both behaviors (`just test-all` with/without the tool) → commit.

## Task 14: final verification

- [ ] `just ci` from `oidtrace/`; `just test-all` if net-snmp present.
- [ ] Validate every line of a CLI-produced trace against the schema (one-liner with
      jsonschema over the glob).
- [ ] `uv run python experiments/trace_format_perf.py` still passes (format unchanged).
