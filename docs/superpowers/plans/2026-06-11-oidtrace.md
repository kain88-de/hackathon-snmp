**Goal:** Implement the OIDTrace CLI (`walk` + `show`) per `docs/superpowers/specs/2026-06-11-oidtrace-architecture-design.md`, emitting traces that conform to `docs/trace-format.md` / `docs/trace-format.schema.json`.

**Architecture:** Hand-rolled minimal BER codec + tolerant decoder; asyncio UDP transport that records everything (wrong request-ids, strays, ICMP errors); sequential walk engine writing gzipped JSONL trace records. **Tech Stack:** Python ≥3.13, stdlib-only runtime (asyncio, gzip, json, argparse). Dev/test: pytest + pytest-asyncio (asyncio_mode=auto), jsonschema, pysnmp (reference decoder), ruff, pyright. uv workspace; `just` targets.

**Out of scope (per spec de-scoping order):** `export-pcap`, SNMP v1, settings-matrix CLI, interactive system-info approval beyond a tty prompt. Violation `missing-end-of-mib` is not emitted in v1 (open enum; a device going silent at tree end is recorded as `end_reason: "unresponsive"` — OIDSense interprets).

## Design notes (read before Task 1)

**Data structures (locked here, used everywhere):**

- `Oid = tuple[int, ...]` — internal OID representation. Strings only at boundaries (CLI args, trace records). Rationale: string comparison orders `"1.3.6.1.10" < "1.3.6.1.2"` — wrong; tuple comparison is correct OID order, needed for loop detection and subtree checks.
- `Varbind(oid: Oid, tag: int, value: bytes)` — frozen dataclass; `vtype` (spec § 5 name) and `vlen` are derived properties. Value bytes exist in memory only; record builders never serialize them.
- `Message(pdu_tag, request_id, f1, f2, varbinds, raw)` | `Malformed(raw, error)` — tolerant decode returns this union; consumers must handle both arms.
- Trace records are **plain dicts** from builder functions in `records.py`. The JSON Schema is the contract; no parallel dataclass hierarchy to drift. Every builder output is schema-validated in tests.
- `Attempt(sent_at, received_at, error)` / `ExchangeIO(attempts, response, strays)` — transport output, maps 1:1 onto the exchange record's wire fields.

**Where we test (observable behavior):**

- **The trace file is the primary observable.** Walker integration tests run a real walk over loopback UDP and assert on the _written trace lines_ (schema-valid, semantically correct) — never on walker internals.
- **Wire bytes are the second contract.** Codec tests are byte-level; encoded packets must decode via pysnmp's spec-driven decoder (`asn1Spec=Message()` — pyasn1's schemaless decoder cannot walk SNMP PDUs; PoC lesson).
- `tests/unit/` = no I/O (oid, ber, codec, violations, records, tracefile). `tests/integration/` = loopback UDP against the quirk emulator. `reference_tools`-marked tests need `snmpbulkwalk`; they skip when absent unless `REQUIRE_REFERENCE_TOOLS=1` (then they fail).
- Timestamps: walker owns `t0 = time.monotonic()`; everything records `round(monotonic() - t0, 6)`.
- **Record sinks**: the walker emits every record through `_Walk.emit()` — gzip file always, plus an optional `on_record` callback (live terminal progress now; an SSE web UI later consumes the same stream).

```python
"""OID value type: tuples of ints internally, strings at boundaries."""

from __future__ import annotations

Oid = tuple[int, ...]


def parse_oid(s: str) -> Oid:
    parts = s.split(".")
    if not parts or any(not p.isdigit() for p in parts):
        raise ValueError(f"not a valid OID: {s!r}")
    return tuple(int(p) for p in parts)


def format_oid(oid: Oid) -> str:
    return ".".join(map(str, oid))


def in_subtree(root: Oid, oid: Oid) -> bool:
    return oid[: len(root)] == root
```

```python
"""Minimal BER primitives — exactly what SNMP v2c needs, nothing more."""

from __future__ import annotations

from oidtrace.oid import Oid


def _length(n: int) -> bytes:
    if n < 0x80:
        return bytes([n])
    body = n.to_bytes((n.bit_length() + 7) // 8, "big")
    return bytes([0x80 | len(body)]) + body


def tlv(tag: int, payload: bytes) -> bytes:
    return bytes([tag]) + _length(len(payload)) + payload


def encode_int(v: int, tag: int = 0x02) -> bytes:
    return tlv(tag, v.to_bytes(v.bit_length() // 8 + 1, "big", signed=True))


def encode_oid(oid: Oid) -> bytes:
    body = bytearray([40 * oid[0] + oid[1]])
    for arc in oid[2:]:
        chunk = [arc & 0x7F]
        arc >>= 7
        while arc:
            chunk.insert(0, (arc & 0x7F) | 0x80)
            arc >>= 7
        body.extend(chunk)
    return tlv(0x06, bytes(body))


def read_tlv(buf: bytes, i: int) -> tuple[int, int, int]:
    """Return (tag, value_start, value_end). Raises ValueError on truncation."""
    if i + 2 > len(buf):
        raise ValueError("truncated TLV header")
    tag = buf[i]
    ln = buf[i + 1]
    i += 2
    if ln & 0x80:
        n = ln & 0x7F
        if n == 0 or i + n > len(buf):
            raise ValueError("bad long-form length")
        ln = int.from_bytes(buf[i : i + n], "big")
        i += n
    if i + ln > len(buf):
        raise ValueError("TLV length exceeds buffer")
    return tag, i, i + ln


def decode_int(body: bytes, *, signed: bool = True) -> int:
    return int.from_bytes(body, "big", signed=signed)


def decode_oid(body: bytes) -> Oid:
    if not body:
        raise ValueError("empty OID body")
    arcs = [body[0] // 40, body[0] % 40] if body[0] < 80 else [2, body[0] - 80]
    acc = 0
    for byte in body[1:]:
        acc = (acc << 7) | (byte & 0x7F)
        if not byte & 0x80:
            arcs.append(acc)
            acc = 0
    if acc:
        raise ValueError("truncated multi-byte arc")
    return tuple(arcs)
```

```python
"""SNMP v2c message codec: encoders + tolerant decoder."""

from __future__ import annotations

from collections.abc import Sequence
from dataclasses import dataclass, field

from oidtrace.ber import decode_int, decode_oid, encode_int, encode_oid, read_tlv, tlv
from oidtrace.oid import Oid

PDU_GET, PDU_GETNEXT, PDU_RESPONSE, PDU_GETBULK = 0xA0, 0xA1, 0xA2, 0xA5

PDU_NAMES = {PDU_GET: "get", PDU_GETNEXT: "getnext", PDU_GETBULK: "getbulk"}

TAG_NAMES = {
    0x02: "Integer", 0x04: "OctetString", 0x05: "Null", 0x06: "ObjectIdentifier",
    0x40: "IpAddress", 0x41: "Counter32", 0x42: "Gauge32", 0x43: "TimeTicks",
    0x44: "Opaque", 0x46: "Counter64",
    0x80: "NoSuchObject", 0x81: "NoSuchInstance", 0x82: "EndOfMibView",
}

EXCEPTION_TAGS = frozenset({0x80, 0x81, 0x82})


def tag_name(tag: int) -> str:
    return TAG_NAMES.get(tag, f"tag:0x{tag:02x}")


def _message(pdu_tag: int, fields: bytes, community: bytes) -> bytes:
    return tlv(0x30, encode_int(1) + tlv(0x04, community) + tlv(pdu_tag, fields))


def _null_varbinds(oids: Sequence[Oid]) -> bytes:
    return tlv(0x30, b"".join(tlv(0x30, encode_oid(o) + tlv(0x05, b"")) for o in oids))


def encode_get(request_id: int, oids: Sequence[Oid], community: bytes = b"public") -> bytes:
    fields = encode_int(request_id) + encode_int(0) + encode_int(0) + _null_varbinds(oids)
    return _message(PDU_GET, fields, community)


def encode_getnext(request_id: int, oid: Oid, community: bytes = b"public") -> bytes:
    fields = encode_int(request_id) + encode_int(0) + encode_int(0) + _null_varbinds([oid])
    return _message(PDU_GETNEXT, fields, community)


def encode_getbulk(
    request_id: int, oid: Oid, non_repeaters: int, max_repetitions: int,
    community: bytes = b"public",
) -> bytes:
    fields = (
        encode_int(request_id) + encode_int(non_repeaters) + encode_int(max_repetitions)
        + _null_varbinds([oid])
    )
    return _message(PDU_GETBULK, fields, community)


def encode_response(
    request_id: int, varbinds: Sequence[tuple[Oid, int, bytes]],
    community: bytes = b"public", error_status: int = 0, error_index: int = 0,
) -> bytes:
    vbs = tlv(0x30, b"".join(
        tlv(0x30, encode_oid(o) + tlv(tag, value)) for o, tag, value in varbinds
    ))
    fields = encode_int(request_id) + encode_int(error_status) + encode_int(error_index) + vbs
    return _message(PDU_RESPONSE, fields, community)
```

```python
@dataclass(frozen=True)
class Varbind:
    oid: Oid
    tag: int
    value: bytes

    @property
    def vtype(self) -> str:
        return tag_name(self.tag)

    @property
    def vlen(self) -> int:
        return len(self.value)


@dataclass(frozen=True)
class Message:
    pdu_tag: int
    request_id: int
    f1: int  # error-status | non-repeaters
    f2: int  # error-index  | max-repetitions
    varbinds: list[Varbind] = field(default_factory=list)
    raw: bytes = b""


@dataclass(frozen=True)
class Malformed:
    raw: bytes
    error: str


def decode_message(raw: bytes) -> Message | Malformed:
    """Tolerant decode: structural garbage becomes Malformed, never an exception."""
    try:
        tag, s, e = read_tlv(raw, 0)
        if tag != 0x30:
            raise ValueError(f"message is not a SEQUENCE (tag 0x{tag:02x})")
        body = raw[s:e]
        _, s, i = read_tlv(body, 0)            # version
        _, s, i = read_tlv(body, i)            # community
        pdu_tag, ps, pe = read_tlv(body, i)
        pdu = body[ps:pe]
        _, s1, j = read_tlv(pdu, 0)
        request_id = decode_int(pdu[s1:j])
        _, s2, j = read_tlv(pdu, j)
        f1 = decode_int(pdu[s2:j])
        _, s3, j = read_tlv(pdu, j)
        f2 = decode_int(pdu[s3:j])
        _, vs, ve = read_tlv(pdu, j)
        vbs_body = pdu[vs:ve]
        varbinds: list[Varbind] = []
        k = 0
        while k < len(vbs_body):
            _, bs, be = read_tlv(vbs_body, k)
            vb = vbs_body[bs:be]
            _, os_, oe = read_tlv(vb, 0)
            vtag, vvs, vve = read_tlv(vb, oe)
            varbinds.append(Varbind(decode_oid(vb[os_:oe]), vtag, vb[vvs:vve]))
            k = be
        return Message(pdu_tag, request_id, f1, f2, varbinds, raw)
    except (ValueError, IndexError) as exc:
        return Malformed(raw, str(exc))


def decode_value(tag: int, value: bytes) -> int | str:
    """Interpret a value — used ONLY for the system-OID allowlist."""
    if tag == 0x02:
        return decode_int(value)
    if tag in (0x41, 0x42, 0x43, 0x46):
        return decode_int(value, signed=False)
    if tag == 0x06:
        from oidtrace.oid import format_oid

        return format_oid(decode_oid(value))
    return value.decode("utf-8", errors="replace")
```

```python
"""Pure violation checks over decoded exchanges (spec § 4.3 vocabulary)."""

from __future__ import annotations

from collections.abc import Sequence

from oidtrace.codec import EXCEPTION_TAGS, Varbind
from oidtrace.oid import Oid


def check_exchange(
    *, sent_id: int, returned_id: int, prev_oid: Oid,
    varbinds: Sequence[Varbind], response_raw: bytes, strays: Sequence[bytes],
) -> list[str]:
    violations: list[str] = []
    if returned_id != sent_id:
        violations.append("request-id-mismatch")
    cur = prev_oid
    for vb in varbinds:
        if vb.tag in EXCEPTION_TAGS:
            continue
        if vb.oid <= cur:
            violations.append("oid-not-increasing")
            break
        cur = vb.oid
    if any(s == response_raw for s in strays):
        violations.append("duplicate-response")
    return violations
```

```python
def header_record(
    *, tool: str, started_at: str, label: str | None, session_id: str,
    run: int, runs_total: int, snmp_version: str, settings: dict[str, Any],

        "started_at": started_at,
        "session": {"id": session_id, "run": run, "runs_total": runs_total},
        "snmp": {"version": snmp_version}, "settings": settings,

    *, seq: int, request: dict[str, Any], attempts: list[dict[str, Any]],
    response_fields: dict[str, int] | None, varbinds: list[Varbind],
    strays: list[dict[str, Any]],
    violations: list[str], malformed: dict[str, Any] | None,
) -> dict[str, Any]:

                           "attempts": attempts}
    if response_fields is not None:
        rec["response"] = {
            **response_fields,
            "varbinds": [{"oid": ".".join(map(str, vb.oid)), "vtype": vb.vtype,
                          "vlen": vb.vlen} for vb in varbinds],
        }
    if strays:
        rec["stray_responses"] = strays
    if violations:
        rec["violations"] = violations
    if malformed is not None:
        rec["malformed"] = malformed
    return rec

    return {"type": "system_info", "at": at, "point": point, "values": values}
```

```python
"""Gzipped JSONL trace I/O: append + flush per record; tolerant truncated reads."""

from __future__ import annotations

import gzip
import json
from collections.abc import Iterator
from pathlib import Path
from types import TracebackType
from typing import Any


class TraceWriter:
    def __init__(self, path: Path) -> None:
        self._f = gzip.open(path, "wt", encoding="utf-8")

    def write(self, record: dict[str, Any]) -> None:
        self._f.write(json.dumps(record, separators=(",", ":")) + "\n")
        self._f.flush()

    def close(self) -> None:
        self._f.close()

    def __enter__(self) -> TraceWriter:
        return self

    def __exit__(self, exc_type: type[BaseException] | None,
                 exc: BaseException | None, tb: TracebackType | None) -> None:
        self.close()


def read_trace(path: Path) -> Iterator[dict[str, Any]]:
    """Yield complete records; stop quietly at a truncation point."""
    try:
        with gzip.open(path, "rt", encoding="utf-8") as f:
            for line in f:
                if not line.endswith("\n"):
                    return  # partial trailing line
                yield json.loads(line)
    except (EOFError, gzip.BadGzipFile):
        return
```

```python
"""Scripted quirk emulator — test fixture and OIDEmu seed.

Responder core with a scripted behavior source (the spec's pluggable-source idea).
Reuses the production codec; deliberately minimal.
"""

from __future__ import annotations

import asyncio
import bisect
from dataclasses import dataclass, field

from oidtrace.codec import encode_response, decode_message, Message
from oidtrace.oid import Oid, parse_oid


@dataclass
class Quirks:
    fixed_request_id: int | None = None
    end_of_mib: str = "eom"          # "eom" | "silence" | "wrap"
    duplicate_responses: bool = False
    slow_prefix: Oid | None = None   # per-OID delay applies under this prefix
    per_oid_delay_s: float = 0.0
    drop_all: bool = False           # never answer (dead device)


@dataclass
class EmuDevice:
    tree: list[tuple[Oid, int, int]]          # (oid, ber_tag, vlen), sorted
    quirks: Quirks = field(default_factory=Quirks)

    @classmethod
    def simple(cls, n_oids: int = 100, quirks: Quirks | None = None) -> EmuDevice:
        cols = [parse_oid(f"1.3.6.1.2.1.2.2.1.{c}") for c in range(1, 11)]
        per_col = n_oids // len(cols) + 1
        tree = sorted(
            ((*col, i), 0x02, 4) for col in cols for i in range(1, per_col + 1)
        )[:n_oids]
        return cls(tree=[(o, t, ln) for o, t, ln in tree], quirks=quirks or Quirks())


class _EmuProtocol(asyncio.DatagramProtocol):
    def __init__(self, device: EmuDevice) -> None:
        self.device = device
        self.keys = [t[0] for t in device.tree]
        self.transport: asyncio.DatagramTransport | None = None

    def connection_made(self, transport) -> None:
        self.transport = transport

    def datagram_received(self, data: bytes, addr) -> None:
        asyncio.get_running_loop().create_task(self._respond(data, addr))

    async def _respond(self, data: bytes, addr) -> None:
        q = self.device.quirks
        if q.drop_all:
            return
        msg = decode_message(data)
        if not isinstance(msg, Message):
            return
        start = msg.varbinds[0].oid
        idx = bisect.bisect_right(self.keys, start)
        count = max(msg.f2, 1)
        chunk = self.device.tree[idx : idx + count]
        if chunk:
            vbs = [(o, tag, b"\x00" * vlen) for o, tag, vlen in chunk]
            delay = sum(
                q.per_oid_delay_s
                for o, _, _ in chunk
                if q.slow_prefix and o[: len(q.slow_prefix)] == q.slow_prefix
            )
            if delay:
                await asyncio.sleep(delay)
        elif q.end_of_mib == "silence":
            return
        elif q.end_of_mib == "wrap":
            o, tag, vlen = self.device.tree[0]
            vbs = [(o, tag, b"\x00" * vlen)]
        else:
            vbs = [(start, 0x82, b"")]
        rid = q.fixed_request_id if q.fixed_request_id is not None else msg.request_id
        raw = encode_response(rid, vbs)
        assert self.transport is not None
        self.transport.sendto(raw, addr)
        if q.duplicate_responses:
            self.transport.sendto(raw, addr)
```

```python
"""UDP transport that records everything and validates nothing (spec: Transport).

Responses are attributed by arrival window, never by request-id. Strictly one
outstanding request. ICMP errors are captured per attempt.
"""

from __future__ import annotations

import asyncio
import errno
from collections.abc import Callable
from dataclasses import dataclass, field

_ERRNO_NAMES = {
    errno.ECONNREFUSED: "icmp-port-unreachable",
    errno.EHOSTUNREACH: "icmp-host-unreachable",
    errno.ENETUNREACH: "icmp-host-unreachable",
}


@dataclass
class Attempt:
    sent_at: float
    received_at: float | None = None
    error: str | None = None


@dataclass
class ExchangeIO:
    attempts: list[Attempt] = field(default_factory=list)
    response: tuple[float, bytes] | None = None     # (received_at, raw)
    strays: list[tuple[float, bytes]] = field(default_factory=list)


class _Protocol(asyncio.DatagramProtocol):
    def __init__(self, rel: Callable[[], float]) -> None:
        self.rel = rel
        self.events: asyncio.Queue[tuple[str, float, bytes | str]] = asyncio.Queue()

    def datagram_received(self, data: bytes, addr) -> None:  # noqa: ARG002
        self.events.put_nowait(("data", round(self.rel(), 6), data))

    def error_received(self, exc: OSError) -> None:
        name = _ERRNO_NAMES.get(exc.errno or 0, "send-failed")
        self.events.put_nowait(("error", round(self.rel(), 6), name))


class UdpTransport:
    def __init__(self, transport: asyncio.DatagramTransport, proto: _Protocol,
                 rel: Callable[[], float]) -> None:
        self._transport = transport
        self._proto = proto
        self._rel = rel

    @classmethod
    async def create(cls, host: str, port: int, rel: Callable[[], float]) -> UdpTransport:
        loop = asyncio.get_running_loop()
        proto = _Protocol(rel)
        transport, _ = await loop.create_datagram_endpoint(
            lambda: proto, remote_addr=(host, port)
        )
        return cls(transport, proto, rel)

    def close(self) -> None:
        self._transport.close()

    async def exchange(self, raw: bytes, *, timeout_s: float, retries: int) -> ExchangeIO:
        io = ExchangeIO()
        for _ in range(retries + 1):
            attempt = Attempt(sent_at=round(self._rel(), 6))
            io.attempts.append(attempt)
            self._transport.sendto(raw)
            try:
                async with asyncio.timeout(timeout_s):
                    kind, at, payload = await self._proto.events.get()
            except TimeoutError:
                continue
            if kind == "error":
                attempt.error = str(payload)
                continue
            attempt.received_at = at
            io.response = (at, bytes(payload))  # type: ignore[arg-type]
            break
        # drain immediately-available extras (duplicates, late replies)
        while not self._proto.events.empty():
            kind, at, payload = self._proto.events.get_nowait()
            if kind == "data":
                io.strays.append((at, bytes(payload)))  # type: ignore[arg-type]
        return io
```

```python
"""Walk engine: drives the GetBulk loop, writes the trace, decides termination."""

from __future__ import annotations

import asyncio
import random
import time
import uuid
from collections import Counter
from collections.abc import Callable
from dataclasses import dataclass
from datetime import UTC, datetime
from pathlib import Path

from oidtrace import codec
from oidtrace.codec import Malformed, Message
from oidtrace.oid import Oid, format_oid, in_subtree, parse_oid
from oidtrace.records import (
    event_record, exchange_record, header_record, summary_record,
)
from oidtrace.tracefile import TraceWriter
from oidtrace.transport import ExchangeIO, UdpTransport

TOOL = "oidtrace 0.1.0"


@dataclass
class WalkSettings:
    bulk_size: int = 10
    timeout_s: float = 2.0
    retries: int = 2
    start_oid: str = "1.3.6.1"
    time_budget_s: float | None = None
    give_up_after: int = 3          # consecutive response-less exchanges before bailing
    community: bytes = b"public"


def _settings_dict(s: WalkSettings) -> dict:
    d = {"bulk_size": s.bulk_size, "timeout_s": s.timeout_s, "retries": s.retries,
         "start_oid": s.start_oid}
    if s.time_budget_s is not None:
        d["time_budget_s"] = s.time_budget_s
    return d


class _Walk:
    def __init__(self, settings: WalkSettings, writer: TraceWriter,
                 on_record: Callable[[dict], None] | None = None) -> None:
        self.s = settings
        self.writer = writer
        self.on_record = on_record
        self.t0 = time.monotonic()
        self.rng = random.Random()
        self.seq = 0
        self.oids_seen: set[Oid] = set()
        self.violation_counts: Counter[str] = Counter()

    def rel(self) -> float:
        return round(time.monotonic() - self.t0, 6)

    def emit(self, record: dict) -> None:
        self.writer.write(record)
        if self.on_record:
            self.on_record(record)

    def write_exchange(self, *, request: dict, io: ExchangeIO,
                       decoded: Message | Malformed | None, violations: list[str]) -> None:
        self.seq += 1
        self.violation_counts.update(violations)
        strays = [{"received_at": at} for at, _raw in io.strays]
        attempts = [
            {k: v for k, v in
             {"sent_at": a.sent_at, "received_at": a.received_at, "error": a.error}.items()
             if not (k == "error" and v is None)}
            for a in io.attempts
        ]
        response_fields = varbinds = malformed = None
        if isinstance(decoded, Message):
            response_fields = {"request_id": decoded.request_id,
                               "error_status": decoded.f1, "error_index": decoded.f2}
            varbinds = decoded.varbinds
        elif isinstance(decoded, Malformed):
            malformed = {"error": decoded.error, "length": len(decoded.raw)}
        self.emit(exchange_record(
            seq=self.seq, request=request, attempts=attempts,
            response_fields=response_fields, varbinds=varbinds or [],
            strays=strays, violations=violations, malformed=malformed,
        ))

    def finish(self, end_reason: str) -> str:
        self.emit(summary_record(
            at=self.rel(), exchanges=self.seq, oids_seen=len(self.oids_seen),
            end_reason=end_reason, violation_counts=dict(self.violation_counts),
        ))
        return end_reason


async def run_walk(
    host: str, port: int, *, settings: WalkSettings, path: Path,
    label: str | None = None, session_id: str | None = None,
    run: int = 1, runs_total: int = 1,
    on_record: Callable[[dict], None] | None = None,
) -> str:
    from oidtrace.violations import check_exchange

    with TraceWriter(path) as writer:
        walk = _Walk(settings, writer, on_record)
        walk.emit(header_record(
            tool=TOOL,
            started_at=datetime.now(UTC).strftime("%Y-%m-%dT%H:%M:%SZ"),
            label=label, session_id=session_id or str(uuid.uuid4()),
            run=run, runs_total=runs_total, snmp_version="2c",
            settings=_settings_dict(settings),
        ))
        transport = await UdpTransport.create(host, port, rel=walk.rel)
        try:
            return await _loop(walk, transport)
        except asyncio.CancelledError:
            walk.finish("interrupted")
            raise
        finally:
            transport.close()


async def _loop(walk: _Walk, transport: UdpTransport) -> str:
    s = walk.s
    root = parse_oid(s.start_oid)
    cur = root
    consecutive_failures = 0
    while True:
        if s.time_budget_s is not None and walk.rel() > s.time_budget_s:
            walk.emit(event_record(at=walk.rel(), kind="time-budget-exceeded"))
            return walk.finish("time-budget-exceeded")
        rid = walk.rng.randrange(1, 2**31)
        if s.bulk_size > 0:
            raw_req = codec.encode_getbulk(rid, cur, 0, s.bulk_size, s.community)
            request = {"pdu": "getbulk", "request_id": rid, "oids": [format_oid(cur)],
                       "non_repeaters": 0, "max_repetitions": s.bulk_size}
        else:
            raw_req = codec.encode_getnext(rid, cur, s.community)
            request = {"pdu": "getnext", "request_id": rid, "oids": [format_oid(cur)]}
        io = await transport.exchange(raw_req, timeout_s=s.timeout_s, retries=s.retries)
        decoded = codec.decode_message(io.response[1]) if io.response else None
        violations: list[str] = []
        if isinstance(decoded, Message):
            violations = check_exchange(
                sent_id=rid, returned_id=decoded.request_id, prev_oid=cur,
                varbinds=decoded.varbinds, response_raw=io.response[1],
                strays=[raw for _, raw in io.strays],
            )
        elif isinstance(decoded, Malformed):
            violations = ["malformed-ber"]
        walk.write_exchange(request=request, io=io, decoded=decoded, violations=violations)
        if not isinstance(decoded, Message):
            consecutive_failures += 1
            if consecutive_failures >= s.give_up_after:
                return walk.finish("unresponsive")
            continue
        consecutive_failures = 0
        data_vbs = [vb for vb in decoded.varbinds if vb.tag not in codec.EXCEPTION_TAGS]
        for vb in data_vbs:
            if in_subtree(root, vb.oid):
                walk.oids_seen.add(vb.oid)
        if any(vb.tag == 0x82 for vb in decoded.varbinds):  # endOfMibView
            return walk.finish("completed")
        if not data_vbs:
            return walk.finish("completed")
        last = data_vbs[-1].oid
        if "oid-not-increasing" in violations:
            walk.emit(event_record(
                at=walk.rel(), kind="oid-loop-detected", detail={"oid": format_oid(last)}
            ))
            return walk.finish("oid-loop")
        if not in_subtree(root, last):
            return walk.finish("completed")
        cur = last
```

```python
"""oidtrace CLI: walk (capture) and show (render)."""

from __future__ import annotations

import argparse
import asyncio
import getpass
import json
import os
import socket
import sys
import uuid
from datetime import UTC, datetime
from pathlib import Path

from oidtrace.tracefile import read_trace
from oidtrace.walker import WalkSettings, run_walk


def _resolve(host: str) -> str:
    """DNS up-front; operator error if it fails (spec: error handling #1)."""
    try:
        return socket.getaddrinfo(host, None, family=socket.AF_INET)[0][4][0]
    except socket.gaierror as exc:
        raise SystemExit(f"oidtrace: cannot resolve {host!r}: {exc}") from exc


def _community() -> bytes:
    env = os.environ.get("OIDTRACE_COMMUNITY")
    if env:
        return env.encode()
    return getpass.getpass("SNMP community: ").encode()


def _cmd_walk(args: argparse.Namespace) -> int:
    try:
        ip = _resolve(args.host)
    except SystemExit as exc:
        print(exc, file=sys.stderr)
        return 2
    out_dir = Path(args.out)
    out_dir.mkdir(parents=True, exist_ok=True)
    stamp = datetime.now(UTC).strftime("%Y%m%dT%H%M%SZ")
    path = out_dir / f"{args.label or 'walk'}-{stamp}.oidtrace.jsonl.gz"
    settings = WalkSettings(
        bulk_size=args.bulk_size, timeout_s=args.timeout, retries=args.retries,
        start_oid=args.start_oid, time_budget_s=args.time_budget,
        community=_community(),
    )
    def progress(rec: dict) -> None:
        if rec.get("type") == "exchange":
            print(f"\r{rec['seq']} exchanges...", end="", file=sys.stderr)

    try:
        end = asyncio.run(run_walk(
            ip, args.port, settings=settings, path=path, label=args.label,
            session_id=str(uuid.uuid4()), hide_system_info=args.hide_system_info,
            on_record=progress,
        ))
        print(file=sys.stderr)
    except KeyboardInterrupt:
        end = "interrupted"
    summary = next((r for r in read_trace(path) if r["type"] == "summary"), None)
    if summary:
        print(f"walk {end}: {summary['exchanges']} exchanges, "
              f"{summary['oids_seen']} OIDs, violations: "
              f"{summary['violation_counts'] or 'none'}")
    print(f"trace written: {path}")
    return 0


def _cmd_show(args: argparse.Namespace) -> int:
    for rec in read_trace(Path(args.trace)):
        print(json.dumps(rec, indent=2))
    return 0


def main(argv: list[str] | None = None) -> int:
    parser = argparse.ArgumentParser(prog="oidtrace")
    sub = parser.add_subparsers(dest="cmd", required=True)

    w = sub.add_parser("walk", help="capture a trace")
    w.add_argument("host")
    w.add_argument("--port", type=int, default=161)
    w.add_argument("--out", default=".", help="output directory")
    w.add_argument("--label", default=None)
    w.add_argument("--bulk-size", type=int, default=10)
    w.add_argument("--timeout", type=float, default=2.0)
    w.add_argument("--retries", type=int, default=2)
    w.add_argument("--start-oid", default="1.3.6.1")
    w.add_argument("--time-budget", type=float, default=None)
    w.add_argument("--hide-system-info", action="store_true")
    w.set_defaults(func=_cmd_walk)

    s = sub.add_parser("show", help="render a trace as JSON")
    s.add_argument("trace")
    s.set_defaults(func=_cmd_show)

    args = parser.parse_args(argv)
    return args.func(args)


if __name__ == "__main__":
    raise SystemExit(main())
```

```python
def _require_tool(name: str) -> None:
    if shutil.which(name):
        return
    if os.environ.get("REQUIRE_REFERENCE_TOOLS"):
        pytest.fail(f"{name} required but not on PATH (REQUIRE_REFERENCE_TOOLS=1)")
    pytest.skip(f"{name} not on PATH — install net-snmp to run reference tests")


async def test_cross_walk_oid_sequences_match(emulator_factory, tmp_path: Path) -> None:
    _require_tool("snmpbulkwalk")
    device = EmuDevice.simple(50)
    async with emulator_factory(device) as (host, port):
        # our walker
        path = tmp_path / "ours.oidtrace.jsonl.gz"
        await run_walk(host, port, settings=WalkSettings(timeout_s=2.0), path=path,
                       hide_system_info=True)
        # reference walker (subprocess; run in executor to not block the loop)
        proc = await asyncio.get_running_loop().run_in_executor(None, lambda: subprocess.run(
            ["snmpbulkwalk", "-v2c", "-c", "public", "-On", "-Cr10",
             f"{host}:{port}", "1.3.6.1"],
            capture_output=True, text=True, timeout=30,
        ))
    ref_oids = [line.split(" ")[0].lstrip(".") for line in proc.stdout.splitlines()
                if line.startswith(".")]
    our_oids = [vb["oid"] for r in read_trace(path)
                if r["type"] == "exchange" and "response" in r
                for vb in r["response"]["varbinds"] if vb["vtype"] != "EndOfMibView"]
    # our walk may overshoot the final bulk past ref's stop point; compare the
    # reference sequence as a prefix of ours, and require full device coverage
    assert ref_oids == our_oids[: len(ref_oids)]
    assert len(set(our_oids)) == 50
```
